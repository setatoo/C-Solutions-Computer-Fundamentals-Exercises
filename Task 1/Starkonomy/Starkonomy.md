# Starkonomy (Bonus Problem)

## Time Limit
1 second

## Memory Limit
256 MB

Stark Industries wants to analyze profit fluctuations.

You are given a sequence of yearly profits.

Find the subsequence that satisfies, in priority order:

1. Strictly increasing
2. Maximum length
3. Maximum average
4. Minimum standard deviation

Subsequence elements do not need to be contiguous.

It is guaranteed that the answer is unique.

---

## Input

First line:
n


Second line:
x1 x2 ... xn

Constraints:
1 ≤ n ≤ 100
1 ≤ xi ≤ 100

---

## Output
Print the selected subsequence.

---

## Example

Input:
7
2 4 6 3 2 4 5

Output:
2 3 4 5
